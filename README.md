# Simply React
> A simple React boilerplate with React Hot Loader.

## Installing / Getting started

Just clone, install, and start!

```shell
git clone https://github.com/satiewaltz/simply-react
cd simply-react/
yarn install
yarn start
```

Open `localhost:8080` to the app.
